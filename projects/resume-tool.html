Resume Compatibility Tool

Overview: A web-based application that analyzes resume and job description alignment for job-seeking veterans.

My Role: I designed and developed both the frontend and backend. I created a keyword scanner that evaluates alignment between uploaded resumes and job descriptions. I also implemented a clean UI, added accessibility features, and created keyword suggestion logic based on common ATS filters. The backend is currently being expanded with Python for deeper analysis.

Tech Used: HTML, CSS, JavaScript, GitHub Pages, Python (in development)

Impact: Helped over 50 veteran job seekers improve resume targeting. Now used by NWVIT in resume prep workshops, helping reduce rejection rates from ATS filters.
